Sections

Sections are just full width container on top level of your body elements tree. There are useful for fluid design layouts but their first goal is to set background and font colors for their child elements.

Default section have vertical padding but you can use bigger or lower padding value or even null.

Section CSS rules enforce color for texts. You can set different font color for paragraph (lists inherit from it), each title level and links.

Location: addons/_sections.scss
